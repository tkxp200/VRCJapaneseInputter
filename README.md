# VRCJapaneseInputter

## 概要

VRChatで日本語入力をサポートするアプリケーションです。
テキストチャット、ウィンドウへのテキスト送信ができます。
動作にはWindows、SteamVRが必須となります。

## 使い方

使い方に関しては(後から追加する)をご覧ください。
特にVRChatのウィンドウにテキストを送信する際は注意が必要です。
